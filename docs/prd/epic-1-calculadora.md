# EPIC-1: Calculadora Web — MVP

**Status:** 📋 Planning
**Owner:** Morgan (@pm)

## Objective

Construir uma calculadora web funcional com as 4 operações básicas, suporte a teclado físico, display de expressão completa, tratamento de erros e acessibilidade — servindo como exercício ponta a ponta do Story Development Cycle no AIOX.

## Business Value

- Demonstra o SDC de 8 passos com um produto real e funcional
- Serve de referência para times novos no AIOX
- Entrega PR aberto no GitHub ao final da sessão

---

## Scope

### In Scope
- Operações: soma, subtração, multiplicação, divisão
- Display duplo (expressão + resultado)
- Suporte a teclado físico (0–9, operadores, Enter, Backspace, Escape)
- Tratamento de erros (divisão por zero)
- Botões C (limpar tudo) e ⌫ (apagar dígito)
- Acessibilidade ARIA (labels, live region, navegação por Tab)
- Testes unitários das funções de cálculo
- Stack: HTML + CSS + JS puro (zero dependências)

### Out of Scope
- Histórico de cálculos (Fase 2)
- Tema claro/escuro (Fase 2)
- Modo científico (Fase 3)
- Deploy / hospedagem

## Success Criteria

- [ ] Calculadora executa as 4 operações corretamente (0 bugs) — validado por testes unitários
- [ ] Teclado físico funciona 100% (dígitos, operadores, Enter, Backspace, Escape)
- [ ] Divisão por zero exibe "Erro: div/0" e recupera com C/Escape
- [ ] Lighthouse Accessibility ≥ 90
- [ ] PR aberto no GitHub com código commitado

---

## Technical Requirements

- HTML5 semântico (`<main>`, `<button>`, `<output>` ou `<div role="status">`)
- CSS custom properties para cores/tamanhos
- JavaScript ES6+ com módulo de lógica separado (`calculator.js`)
- Vitest (ou script de teste standalone) para cobertura unitária
- Sem bundler obrigatório — arquivos estáticos servíveis diretamente
